搭建博客记

发表于2025-02-06|更新于2025-02-08
|浏览量:

前提

我的电脑是22年发布的MacBook Pro，芯片为Apple M2，我用该设备进行GitHub博客的搭建。这篇笔记我将边搭建博客边记录，主要记录我在搭建过程中遇到的技术问题（有的问题可能会很蠢）。如果我成功搭建了博客，它将成为我的第一篇博文。我希望这篇笔记能被我成功发表吧。

遇到的问题

Homebrew因为网速问题一直无法下载

我尝试了各种梯子，最终都没办法下下来，最后查询一个网站解决该问题。

“zsh: command not found: nvm“

一开始我一直以为是因为我没有安装成功nvm，通过查阅资料，我发现需要在.zshrc文件中添加nvm的相关代码才能正确运行nvm，这一步骤叫做配置环境变量。

环境变量类似于“程序”手中的说明书，“程序”通过环境变量的指导来完成运行。

编辑.zshrc文件并不是一帆风顺

首先，编辑.zshrc文件的时候要以管理员的身份打开，否砸不能进行修改。
相关代码如下：
sudo nano ~/.zshrc

其次，我并不是文件的所有者，在我成为所有者之后，还需要让我拥有对.zshrc文件具有“读写”的权限，相关代码如下：

修改当前用户为所有者：sudo chown $(whoami) ~/.zshrc

让我自己拥有读写的权限：chmod u+rw ~/.zshrc

node的版本选择

我打算直接用
brew install node
来安装node，因为这样下载的是最新最稳定的node版本。但是这个具有限制性，这个最新的版本指的是当前”Homebrew下最新的版本。我目前搭建博客只是想发点自己的笔记，应该不会有其他的用途，所以先凑合着用，后续需要改变再说。

zsh: command not found: hexo

没事了，这一次应该是网速问题导致下载下来的hexo不完整。切换成淘宝的镜像源之后再下载就好了。

hexo d后报错Spawn failed

我日了，这我解决了一天没解决成，我一个一个排查。很烦！

网上搜到的大部分教程不会说很细的这步，现在 GitHub不能用密码登陆了，在终端输入的时候要使用token。
